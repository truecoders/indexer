# 🔍 Индексатор Файлов (File Indexer)

Высокопроизводительное десктопное приложение нового поколения для мгновенного локального полнотекстового поиска по содержимому и именам документов с интеллектуальным отслеживанием изменений в реальном времени.

Приложение полностью конфиденциально: вся обработка, парсинг файлов и база данных SQLite находятся исключительно на вашем локальном устройстве.

---

## 🚀 Ключевые преимущества

- **Молниеносный полнотекстовый поиск**: Базируется на модуле **SQLite FTS5**. Результаты ранжируются по релевантности (`rank`) с мгновенным выделением совпадений.
- **Интегрированный Split View предпросмотр**: Клик по файлу открывает встроенную правую панель с текстовым содержимым документа. Найденные слова подсвечиваются с возможностью навигации по ним (кнопки `↑` / `↓` или горячие клавиши `F3` / `Shift+F3`) — больше не нужно открывать Word или Excel ради одной строчки.
- **Автоматическое отслеживание изменений (File Watcher)**: Фоновый модуль на базе библиотеки `notify` на Rust мгновенно реагирует на добавление, переименование, изменение или удаление файлов в отслеживаемых папках и обновляет поисковый индекс «на лету».
- **Инкрементальный парсинг**: Индексатор сканирует только новые или изменённые файлы (проверка по времени модификации), экономя заряд батареи и ресурсы процессора.
- **Поддержка популярных форматов**:
  - **Документы Word** (`.docx` — прямой парсинг XML-структуры без необходимости наличия установленного MS Office)
  - **Таблицы Excel** (`.xlsx`, `.xls` — сверхбыстрое чтение ячеек на базе парсера Calamine)
  - **Текстовые форматы** (`.txt`, `.md`, `.csv` — с автоматическим определением кодировок Windows-1251 и UTF-8)

## ✨ Умные фишки и UX-функции

- **Персистентная история поиска в БД**: Ваши последние 20 поисков автоматически сохраняются в базу данных со всеми параметрами (выбранная папка, расширения, режим поиска, вид и направление сортировки). Клик по записи в сайдбаре мгновенно восстанавливает весь контекст поиска.
- **Мгновенная интерактивная фильтрация**: Основной поиск запускается осознанно по нажатию кнопки **«Найти»** или клавиши **Enter** (никаких ложных срабатываний и тормозов при вводе запроса). Но если результаты уже найдены, изменение любого фильтра или сортировки перестраивает результаты **мгновенно и без задержек**.
- **Интерактивное управление исключениями (TagsInput)**: Каждая папка при добавлении автоматически защищается от сканирования системного мусора по умолчанию (`node_modules`, `.git`, `~$*`, `*.tmp`). Вы можете настраивать эти исключения тегами прямо в карточке папки, удаляя старые или добавляя свои паттерны.
- **Статистика в реальном времени**: Встроенный в сайдбар аналитический блок показывает общее число отслеживаемых папок, успешно проиндексированных файлов, общее количество слов в индексе и живой индикатор текущего процесса сканирования.
- **Удобные горячие клавиши**: Нажатие `Ctrl + F` в любой вкладке мгновенно фокусирует курсор на строке поиска.

## 🛠️ Технологический стек

- **Backend**: Rust, Tauri v2, Rusqlite (SQLite + FTS5), Calamine (Excel parser), Notify (fs watcher).
- **Frontend**: React 19, TypeScript, Mantine UI v9 (с кастомной 10-оттеночной оливковой темой), Tabler Icons.
- **Компиляция**: Standalone-сборка под Windows (собирается в один компактный `.exe` файл размером ~12.5 МБ).

## 💻 Разработка и сборка

### Требования
- Установленный [Rust (rustup)](https://rustup.rs/)
- Установленный [Node.js](https://nodejs.org/)

### Запуск в режиме разработки
```bash
# Установка NPM-зависимостей
npm install

# Запуск Tauri-приложения в режиме Live-Reload
npm run tauri dev
```

### Сборка standalone-версии (`Indexer.exe`)
Сборка компилирует оптимизированный бинарный файл на Rust, упаковывает фронтенд и копирует готовый исполняемый файл в корневую директорию проекта:
```bash
npm run build:exe
```
После завершения в корне проекта появится файл `Indexer.exe`, готовый к запуску и переносу на любой компьютер.
